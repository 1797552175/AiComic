## 一、核心差异化定位

### 我们 vs 竞品

| 维度 | Runway/Pika | 剪映/腾讯智影 | 白日梦/海螺 | **我们的产品** |
|------|-------------|---------------|-------------|----------------|
| 漫画原生逻辑 | ❌ 视频逻辑 | ❌ 剪辑逻辑 | ⚠️ 弱 | ✅ **以漫画创作为原点** |
| 角色一致性 | ❌ 每次随机 | ❌ 无 | ⚠️ 弱 | ✅ **角色ID锁定** |
| 分镜可控性 | ⚠️ 粗粒度 | ✅ 时间线 | ❌ 无 | ✅ **精细化分镜控制** |
| 动态漫"微动效" | ❌ 过度动画化 | ⚠️ 模板固定 | ❌ 无 | ✅ **保留漫画质感** |
| 中文TTS | ⚠️ 需第三方 | ✅ 内置 | ✅ 内置 | ✅ **多角色+情感TTS** |
| 一站式工作流 | ⚠️ 需串联 | ⚠️ 需串联 | ⚠️ 半自动 | ✅ **剧本→导出 全自动** |

## 二、产品使用流程

```
┌─────────────────────────────────────────────────────────────┐
│                      用户核心路径                           │
└─────────────────────────────────────────────────────────────┘

  ① 输入剧本                                                     
     ↓
  ② AI智能解析（场景拆分/角色识别/情绪标注）                       
     ↓
  ③ 可视化分镜编辑器（拖拽调整/镜头类型/时长）                     
     ↓
  ④ 角色库管理（上传参考图/锁定角色形象）                          
     ↓
  ⑤ 批量画风选择（一键应用风格/单镜微调）                          
     ↓
  ⑥ AI画风迁移（统一保持漫画质感）                                
     ↓
  ⑦ 动态化处理（镜头运动/口型同步/微动效）                          
     ↓
  ⑧ 配音配乐（多角色TTS/智能BGM）                                 
     ↓
  ⑨ 预览确认                                                     
     ↓
  ⑩ 导出发布（多格式/多分辨率/封面字幕）
```

## 三、模块详细设计

### 3.1 剧本输入模块

**输入方式：**

| 方式 | 说明 | 适合场景 |
|------|------|----------|
| 自由文本 | 富文本框，手动换行分段 | 快速创作 |
| 对话格式 | `角色名：对话内容` 自动识别 | 多角色对话 |
| 上传文件 | TXT/Word/PDF 批量导入 | 已有剧本 |
| AI扩写 | 输入30字剧情概要，AI扩写500字 | 没有完整剧本 |
| 参考风格 | 粘贴参考文本，AI学习风格 | 系列作品 |

**AI解析能力：**

```
输入剧本 → 输出结构化数据
{
  "场景列表": [
    {
      "场景名": "咖啡馆·白天",
      "场景描述": "温馨的咖啡馆，窗外阳光明媚",
      "镜头列表": [
        {
          "镜头ID": "shot_001",
          "镜头类型": "中景",
          "时长": 3,
          "角色": ["小明", "小红"],
          "对话": "小明：这家店真不错！",
          "情绪": "轻松愉快",
          "画面关键词": "咖啡馆内景，阳光，暖色调，两位年轻人对话"
        }
      ]
    }
  ],
  "角色列表": [
    {"名字": "小明", "性别": "男", "年龄": "青年", "特征": "短发，戴眼镜"},
    {"名字": "小红", "性别": "女", "年龄": "青年", "特征": "长发，穿裙子"}
  ]
}
```

**智能检测：**
- 🔍 角色检测：自动识别新角色，提示是否加入角色库
- 🎭 情绪识别：对话→情绪标签（高兴/悲伤/愤怒/惊讶/平静）
- 📝 分镜建议：自动拆分合适长度的镜头
- ✍️ 格式优化：口语化→戏剧化表达一键转换

### 3.2 分镜编辑器

**核心交互：时间线拖拽式**

```
┌────────────────────────────────────────────────────────────────────┐
│  分镜时间线                                                          │
├────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  场景1：咖啡馆                                        场景2：街道      │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐                      ┌─────┐     │
│  │ 镜1 │→│ 镜2 │→│ 镜3 │→│ 镜4 │                      │ 镜5 │     │
│  │ 3s  │ │ 4s  │ │ 2s  │ │ 3s  │                      │ 5s  │     │
│  └─────┘ └─────┘ └─────┘ └─────┘                      └─────┘     │
│                                                                     │
│  ▶ 拖拽调整顺序  │  + 添加镜头  │  - 删除  │  ↵ 拆分镜头            │
└────────────────────────────────────────────────────────────────────┘
```

**每个镜头的可编辑属性：**

| 属性 | 说明 | 操作 |
|------|------|------|
| 镜头类型 | 近景/远景/特写/全景/过肩/POV | 下拉选择 |
| 时长 | 1-10秒 | 拖拽或输入 |
| 角色 | 出现在本镜的角色 | 多选 |
| 对话 | 角色对话/旁白 | 文本输入 |
| 画面描述 | AI生图关键词 | 自动生成+手动修改 |
| 镜头运动 | 推/拉/摇/移/固定 | 下拉选择 |
| 情绪氛围 | 影响配乐和色调 | 标签选择 |

**镜头类型图示：**

```
🌅 远景(Establishing)  → 建立场景，展示全貌
🚶 全景(Full Shot)      → 角色全身
💬 中景(Medium Shot)    → 腰部以上，对话常用
😊 特写(Close-up)       → 面部/物品细节
👥 双人镜头(Two Shot)   → 两人同框
🔒 过肩(Over Shoulder)  → 代入感强
🎬 POV(主观镜头)        → 观众视角
🔍 特技(Reaction Shot)  → 反应镜头
```

### 3.3 角色一致性系统（核心竞争力）

**角色库管理：**

```
┌─────────────────────────────────────────────────────────────┐
│  角色库                                                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐        │
│  │ 角色卡片 │  │ 角色卡片 │  │ 角色卡片 │  │  + 新建 │        │
│  │  小明   │  │  小红   │  │  旁白   │  │  角色   │        │
│  │ [头像]  │  │ [头像]  │  │ [默认]  │  │         │        │
│  └─────────┘  └─────────┘  └─────────┘  └─────────┘        │
│                                                             │
│  角色详情                                                    │
│  ────────────────────────────────────────                   │
│  姓名：小明                                                  │
│  性别：男                                                    │
│  年龄：青年（20-30岁）                                       │
│  外貌特征：短发、戴眼镜、身高178cm                           │
│  性格：温和、有点腼腆                                        │
│  服装风格：休闲装、白色T恤、深色牛仔裤                       │
│                                                             │
│  参考图：                                                    │
│  ┌─────┐ ┌─────┐ ┌─────┐                                    │
│  │正脸 │ │侧脸 │ │表情 │  [+ 上传更多]                      │
│  │     │ │     │ │参考 │                                    │
│  └─────┘ └─────┘ └─────┘                                    │
│                                                             │
│  ⚠️ 至少上传3张不同角度的参考图以确保一致性                    │
└─────────────────────────────────────────────────────────────┘
```

**一致性保障技术：**

| 技术 | 说明 |
|------|------|
| 角色ID嵌入 | 每角色分配唯一ID，贯穿全片 |
| 参考图提取特征 | 上传3-5张参考，提取面部/服装特征 |
| LoRA微调 | 可选：基于参考图微调模型，提升一致性 |
| 服装一致性 | 服装颜色/款式跨镜头保持一致 |
| 多角度生成 | 系统自动补充角色不同角度参考 |

**角色绑定工作流：**

```
分镜识别角色 → 提示绑定角色ID → 选择角色库已有角色或新建
     ↓
自动从角色库提取参考图特征
     ↓
画面生成时注入角色特征
     ↓
输出保持角色一致性的镜头
```

### 3.5 动态化模块

**镜头运动预设（核心差异化）：**

动态漫的精髓是"微动效"——不是让角色完全动起来，而是通过镜头运动和局部动效让画面"活"起来。

```
┌─────────────────────────────────────────────────────────────┐
│  镜头运动                                                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  固定镜头   ➖  静止画面，适合对话/特写                        │
│                                                             │
│  横向移动                                                    │
│  → 从左向右平移    ← 从右向左平移    ↔  左右振荡              │
│                                                             │
│  纵向移动                                                    │
│  ↑ 上摇（抬头效果）  ↓ 下摇（低头效果）  ↕ 上下振荡            │
│                                                             │
│  推进拉远                                                    │
│  🔍 推进（Zoom In）— 强调/聚焦    📤 拉远（Zoom Out）— 展示   │
│                                                             │
│  旋转效果                                                    │
│  🔄 轻微旋转（手持感）    ↻ 360°旋转                        │
│                                                             │
│  复合运动                                                    │
│  ⤵️ 推进+平移组合    ⤴️ 拉远+摇镜组合                       │
│                                                             │
│  运动速度：[慢速 0.5x] [正常 1x] [快速 2x]                   │
└─────────────────────────────────────────────────────────────┘
```

**自动口型同步：**

```
输入对话文本 → AI分析音素 → 自动生成口型动画

支持：中文（普通话/粤语/方言）/ 英语 / 日语 / 韩语
口型精度：音素匹配（不是简单的张嘴闭嘴）
```

**微动效预设：**

| 效果 | 说明 | 适用场景 |
|------|------|----------|
| 头发飘动 | 轻微飘动，2-3秒循环 | 户外/有风 |
| 衣物摆动 | 裙摆/衣角微动 | 行走/站立 |
| 呼吸效果 | 胸腔轻微起伏 | 静止对话 |
| 表情微动 | 眨眼/微笑/皱眉 | 情绪镜头 |
| 背景微动 | 云/树叶/水波 | 场景氛围 |
| 速度线 | 运动物体两侧 | 速度感 |

**转场效果：**

| 转场 | 说明 |
|------|------|
| ⚫ 黑场淡入淡出 | 场景切换，时间流逝 |
| ⚪ 白场淡入淡出 | 梦境/回忆 |
| 🔲 叠化 | 平滑过渡 |
| 📖 翻页 | 漫画感 |
| 💨 闪白 | 强冲击感 |
| ⏩ 跳切 | 快节奏 |

## 五、竞品差异化总结

### 相比 Runway / Pika Labs

| 优势 | 说明 |
|------|------|
| 🎯 漫画原生 | 不是通用视频生成，是专门为漫画创作者设计的工作流 |
| 🎭 角色一致性 | 内置角色库系统，解决AI生成角色不一致的核心痛点 |
| 📖 分镜控制 | 精细到每个镜头的类型/时长/运动控制 |
| 📱 中文友好 | 中文TTS、中文UI、中文创作习惯适配 |

### 相比国内剪映 / 腾讯智影

| 优势 | 说明 |
|------|------|
| 🤖 AI原生 | 不是剪辑工具加AI，是从AI创作出发的全新工作流 |
| 🎨 画风多样 | 不仅仅有模板，而是真正的AI画风生成 |
| 🔧 可定制性 | 每个环节都可精细调整，不是黑箱 |
| 📦 独立性 | 不依赖第三方剪辑软件，独立完成全流程 |

### 相比白日梦 / 海螺AI

| 优势 | 说明 |
|------|------|
| 🎬 真正的动态漫 | 不是静态图+配乐，是有镜头运动和动效的动态漫 |
| ✂️ 分镜可控 | 用户可以控制每个镜头的表达 |
| 🎭 角色一致性 | 有完整的角色库系统 |
| 📚 完整工作流 | 从剧本到导出的一站式解决方案 |
