# 快捷键与操作辅助原型

> 版本：v1.0 | 日期：2026-03-24 | 状态：待评审

---

## 一、背景与目标

参考Figma、Photoshop的快捷键设计，提升高级用户的操作效率。

### 目标
- 减少鼠标操作
- 提升专业用户效率
- 提供操作引导

---

## 二、功能清单

| 序号 | 功能点 | 描述 | 优先级 |
|------|--------|------|--------|
| 1 | 全局快捷键 | 全局通用操作 | P1 |
| 2 | 上下文快捷键 | 当前页面特定操作 | P1 |
| 3 | 操作引导 | 新手快捷键提示 | P2 |
| 4 | 自定义快捷键 | 用户自定义快捷键 | P2 |

---

## 三、快捷键设计

### 3.1 全局快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| Ctrl/Cmd + K | 全局搜索 | 打开搜索框 |
| Ctrl/Cmd + N | 新建 | 新建角色/作品 |
| Ctrl/Cmd + S | 保存 | 保存当前内容 |
| Ctrl/Cmd + Z | 撤销 | 撤销上一步 |
| Ctrl/Cmd + Shift + Z | 重做 | 重做 |
| ? | 帮助 | 显示快捷键列表 |

### 3.2 角色管理快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| Ctrl/Cmd + C | 复制角色 | 复制选中角色 |
| Ctrl/Cmd + V | 粘贴角色 | 粘贴角色 |
| Delete | 删除角色 | 删除选中角色 |
| E | 编辑 | 编辑选中角色 |
| Space | 预览 | 预览选中角色 |
