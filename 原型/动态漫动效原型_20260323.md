# 动态漫动效原型

> 日期：2026-03-23 | 版本：v1.0 | 状态：初稿
> 参考：竞品分析报告_20260322.md

---

## 一、产品概述

### 1.1 核心问题

竞品（Pika/Runway）要么无动态（静态漫画工具），要么"过度动画化"（AI自动让图片完全动起来，失去漫画质感）。用户需要一个保留漫画美感的"动态漫"工具，而非"会动的视频"。

### 1.2 解决方案

定义**动态漫标准动效体系**：镜头运动预设 + 口型同步 + 微动效叠加，让画面有生命感但不失去漫画质感。

---

## 二、动态漫动效体系

### 2.1 动效分类

| 类型 | 说明 | 强度 | 适用场景 |
|------|------|------|----------|
| **镜头运动** | 画面整体的运镜效果 | 中 | 所有镜头 |
| **微动效** | 画面元素的轻微动态 | 轻 | 人物、自然元素 |
| **口型同步** | 角色说话时的口型动画 | 中 | 有台词的镜头 |
| **转场动效** | 镜头间的过渡效果 | 重 | 分镜连接处 |

### 2.2 设计原则

1. **轻量化优先**：动效服务于叙事，不喧宾夺主
2. **保留漫画质感**：避免过度动画化，保持漫画的"静中有动"
3. **可调节**：用户可控制动效强度（0-100%）
4. **预设+自定义**：提供预设，也支持高级用户自定义

## 三、镜头运动预设

### 3.1 预设类型

| 运动类型 | 描述 | 参数 | 适用场景 |
|----------|------|------|----------|
| **固定** | 画面静止不动 | 无 | 对话、静态场景 |
| **推进** | 镜头向前推进 | 速度：慢/中/快 | 聚焦角色、强调情绪 |
| **拉远** | 镜头向后拉远 | 速度：慢/中/快 | 展示全貌、转场 |
| **左摇** | 镜头向左摇动 | 速度、角度 | 扫视场景 |
| **右摇** | 镜头向右摇动 | 速度、角度 | 扫视场景 |
| **上摇** | 镜头向上摇动 | 速度、角度 | 仰视角色/建筑 |
| **下摇** | 镜头向下摇动 | 速度、角度 | 俯视全景 |
| **环绕** | 镜头围绕中心旋转 | 速度、方向 | 展示立体感 |
| **跟随** | 镜头跟随角色移动 | 目标角色ID | 角色行走/奔跑 |
| **震动** | 轻微镜头抖动 | 强度：轻/中/重 | 冲击感、紧张感 |

### 3.2 镜头运动参数面板

```
┌─────────────────────────────────────────┐
│ 镜头运动设置                            │
├─────────────────────────────────────────┤
│ 运动类型：[推进           ▼]            │
│                                         │
│ 速度：    ○慢  ●中  ○快                 │
│                                         │
│ 起始位置：[═══●═══════] 30%             │
│ 结束位置：[════════●═══] 80%             │
│                                         │
│ 预览：[ ▶ 播放 ]                         │
│ 强度：  [══════════●] 100%              │
└─────────────────────────────────────────┘
```

### 3.3 镜头运动实现原理

```
镜头生成完成
    ↓
加载镜头运动预设参数
    ↓
计算关键帧：
  - 第0帧：起始画面
  - 第N帧：中间过渡（AI生成插帧）
  - 最后帧：结束画面
    ↓
输出带镜头运动的视频片段
```

## 四、口型同步

### 4.1 功能描述

根据台词内容，自动生成角色口型动画，无需手动K帧。

### 4.2 支持的音视频输入

| 输入类型 | 处理方式 |
|----------|----------|
| TTS生成音频 | 从音轨提取音素，直接映射口型 |
| 用户上传音频 | ASR识别音素 → 映射口型 |
| 角色静音 | 不生成口型动效 |

### 4.3 口型类型映射

| 口型类型 | 代表音素 | 适用场景 |
|----------|----------|----------|
| **闭唇** | 休止、m/b/p | 句末、停顿 |
| **微张** | i/y | 短促音 |
| **张开** | a/o/e | 大部分元音 |
| **圆唇** | u/o | o/u音 |
| **咧嘴** | s/z/c | s/z/c音 |

### 4.4 口型同步参数面板

```
┌─────────────────────────────────────────┐
│ 口型同步设置                            │
├─────────────────────────────────────────┤
│ 口型模式：[自动识别     ▼]              │
│  （自动/闭唇/微张/张开/圆唇）            │
│                                         │
│ 强度：  [═════════●──] 85%              │
│  （数值越低口型越轻微）                  │
│                                         │
│ 对齐精度：[高（推荐）  ▼]               │
│  （高/中/低）                            │
│                                         │
│ [▶ 预览口型动画]                        │
└─────────────────────────────────────────┘
```

### 4.5 口型同步实现流程

```
音频输入
    ↓
ASR音素识别
    ↓
时间轴对齐（音频ms → 口型帧）
    ↓
口型序列生成（每帧对应一个口型类型）
    ↓
角色面部动效渲染
    ↓
叠加到原画面
```

## 五、微动效叠加

### 5.1 微动效类型

| 动效名称 | 描述 | 强度 | 适用对象 |
|----------|------|------|----------|
| **头发飘动** | 发丝轻微飘动 | 轻 | 角色 |
| **呼吸起伏** | 胸部/腹部轻微起伏 | 轻 | 角色 |
| **表情微动** | 眉毛、嘴角轻微变化 | 轻 | 角色 |
| **衣物摆动** | 服装边缘轻微摆动 | 中 | 角色 |
| **眼睛眨眼** | 自然的眨眼动作 | 轻 | 角色 |
| **背景微动** | 云/草/树叶轻微晃动 | 轻 | 背景 |
| **光影流动** | 光影位置的轻微变化 | 轻 | 整体画面 |
| **特效叠加** | 火花/星光/雾气 | 中 | 整体/局部 |

### 5.2 微动效预设包

**预设1：自然呼吸**
- 呼吸起伏：开（强度30%）
- 眼睛眨眼：开（自然频率）
- 背景微动：关

**预设2：动感少年**
- 头发飘动：开（强度50%）
- 衣物摆动：开（强度40%）
- 表情微动：开（强度30%）

**预设3：静谧氛围**
- 光影流动：开（强度20%）
- 背景微动：开（强度15%）
- 呼吸起伏：开（强度15%）

**预设4：战斗动感**
- 头发飘动：开（强度80%）
- 衣物摆动：开（强度70%）
- 震动：开（强度30%）

### 5.3 微动效参数面板

```
┌─────────────────────────────────────────────────────────┐
│ 微动效叠加                                      [预设 ▼]│
├─────────────────────────────────────────────────────────┤
│ 动效类型              开关   强度      备注            │
│ ─────────────────────────────────────────────────────  │
│ 头发飘动              [●──]  [══●═══]  30%             │
│ 呼吸起伏              [●──]  [═══●══]  20%             │
│ 表情微动              [●──]  [═●════]  15%             │
│ 衣物摆动              [──●]  [═════●═]  关闭           │
│ 眼睛眨眼              [●──]  [自动]     正常频率       │
│ 背景微动              [──●]  [═════●═]  关闭           │
│ 光影流动              [──●]  [═════●═]  关闭           │
│ 特效叠加              [──●]  [无]       未选择         │
│                                                         │
│ 预览：[ ▶ 预览当前镜头微动效]                            │
└─────────────────────────────────────────────────────────┘
```

## 六、转场动效

### 6.1 转场类型

| 转场类型 | 描述 | 时长 | 适用场景 |
|----------|------|------|----------|
| **无（硬切）** | 直接切换 | 0s | 快节奏、战斗 |
| **黑场** | 黑屏过渡 | 0.3-0.5s | 场景转换 |
| **白场** | 白屏过渡 | 0.3-0.5s | 梦境/回忆 |
| **叠化** | 前后画面重叠 | 0.5-1s | 缓慢过渡 |
| **滑动** | 画面滑动进入 | 0.3-0.5s | 同一场景位置切换 |
| **缩放** | 前后画面缩放过渡 | 0.3-0.5s | 聚焦/远景切换 |
| **漫画式** | 速度线+画面切换 | 0.2-0.3s | 漫画原生效果 |
| **震动转场** | 画面震动后切换 | 0.3s | 冲击感转场 |

## 七、时间线编辑器

### 7.1 时间线布局

```
┌─────────────────────────────────────────────────────────┐
│ 工具栏：[保存] [撤销] [重做] [预览] [导出]              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  0s    3s    6s    9s    12s   15s   18s   21s         │
│  |     |     |     |     |     |     |     |           │
│ ┌─────┐┌─────┐┌─────┐┌─────┐┌─────┐┌─────┐           │
│ │ 分镜1 │→│ 分镜2 │→│ 分镜3 │→│ 分镜4 │→│ 分镜5 │           │
│ │[推进] │ │[固定] │ │[拉远] │ │[震动] │ │[左摇] │           │
│ └─────┘└─────┘└─────┘└─────┘└─────┘└─────┘           │
│                                                         │
├─────────────────────────────────────────────────────────┤
│ 当前选中：分镜 2                                        │
│ ┌─────────────────────────────────────────────────────┐│
│ │ 角色：小明  台词：「今天天气真好」                    ││
│ │ 镜头运动：[固定        ▼]  强度：[═══●═══] 100%     ││
│ │ 微动效：[自然呼吸预设 ▼]                             ││
│ │ 口型：[自动识别      ▼]                              ││
│ │ 转场→下一：[黑场 0.4s]                              ││
│ └─────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────┘
```

### 7.2 时间线操作

- **拖拽调整时长**：拖动分镜边缘
- **拖拽调整顺序**：拖动分镜到目标位置
- **双击预览**：双击分镜进入预览模式
- **右键菜单**：复制/粘贴/删除/插入分镜
- **批量操作**：多选分镜批量设置动效

## 八、动效强度控制

### 8.1 全局动效强度

```
┌─────────────────────────────────────────┐
│ 动态漫强度        [═●════════] 70%       │
├─────────────────────────────────────────┤
│ 预设：                                     │
│ ○ 原画风（0%）— 完全静态漫画              │
│ ● 轻度（50-70%）— 微动感，推荐           │
│ ○ 中度（70-85%）— 明显动态               │
│ ○ 动画化（85-100%）— 接近动画            │
└─────────────────────────────────────────┘
```

### 8.2 强度影响说明

| 强度 | 镜头运动 | 微动效 | 口型 |
|------|----------|--------|------|
| 0% | 无 | 无 | 无 |
| 30% | 慢速 | 极轻微 | 轻微 |
| 50% | 正常 | 轻微 | 正常 |
| 70% | 正常+增强 | 中等 | 明显 |
| 85% | 明显 | 较强 | 夸张 |
| 100% | 极限 | 极限 | 夸张 |
