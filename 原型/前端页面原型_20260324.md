# AI 创作动态漫 - 前端页面原型

**版本：** v1.0
**日期：** 2026-03-24
**功能：** 基于竞品 ComicAI，实现核心功能的前端页面

---

## 1. 技术栈

- **前端框架：** React 18 + Vite
- **UI 库：** Ant Design 5
- **状态管理：** Zustand
- **HTTP 客户端：** Axios
- **样式：** TailwindCSS
- **路由：** React Router v6

---

## 2. 核心页面

### 2.1 登录/注册页
- 手机号 + 验证码登录
- 用户名 + 密码注册
- 忘记密码

### 2.2 项目列表页
- 我的项目（卡片展示）
- 创建新项目按钮
- 项目搜索/筛选
- 项目状态标签（草稿/生成中/已完成）

### 2.3 故事编辑器页（核心）
- 故事输入框（输入一句话生成故事）
- AI 扩展的故事脚本展示
- 分镜列表（每个分镜可编辑）
- 角色管理面板
- 风格选择（写实/动漫/厚涂/水墨）
- 尺寸选择（16:9/9:16/1:1）
- 生成按钮

### 2.4 分镜详情页
- 单个分镜的编辑界面
- 角色动作/表情描述
- 对话/旁白输入
- AI 生成图片预览
- AI 动态化设置
- 配音设置

### 2.5 角色管理页
- 角色列表（头像 + 名称）
- 创建角色（上传参考图）
- 角色详情编辑
- 一致性评分显示

### 2.6 作品预览页
- 漫画分镜预览
- 动态播放
- 配音播放
- 导出选项

## 3. API 对接

### 3.1 后端 API 端点
```
POST /api/auth/login      - 登录
POST /api/auth/register   - 注册
GET  /api/projects       - 获取项目列表
POST /api/projects       - 创建项目
GET  /api/projects/:id   - 获取项目详情
PUT  /api/projects/:id   - 更新项目
DELETE /api/projects/:id - 删除项目
POST /api/story/generate  - AI 生成故事
POST /api/image/generate  - AI 生成图片
POST /api/video/animate  - AI 生成动态
```

### 3.2 前端配置
```javascript
const API_BASE = 'http://150.109.243.164:8000/api/v1'
```

## 5. 验收标准

1. 用户可以注册/登录
2. 可以创建、编辑、删除项目
3. 可以输入故事并获得 AI 扩展
4. 可以查看和管理角色
5. 可以生成图片和动态
6. 前后端 API 联通
7. 响应式布局（PC + 移动端）
