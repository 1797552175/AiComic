# AiComic API 开放平台

让AI漫画能力接入你的应用。

## 支持接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/characters | GET/POST | 角色管理 |
| /api/v1/comics/generate | POST | 漫画生成 |
| /api/v1/tts | POST | 语音合成 |

## 快速开始

```python
import aicomic

client = aicomic.Client(api_key="your_key")

# 创建角色
character = client.characters.create(
    reference_image="https://example.com/image.jpg"
)

# 生成漫画
comic = client.comics.generate(
    character_id=character.id,
    scene="在咖啡馆相遇"
)
```
