## 三、核心卖点

### 3.1 主打卖点

**「上传一张图，AI读懂你的角色，然后用一辈子」**

| 卖点 | 说明 | 竞品对比 |
|------|------|----------|
| **一键角色卡** | 上传参考图，AI自动提取特征生成角色描述 | Runway/Pika ❌ 无 |
| **LoRA + IP-Adapter** | 双技术保障，一致性≥90% | 竞品只用LoRA，~60% |
| **完整角色库** | 搜索/筛选/分类，支持角色变体 | 竞品❌ 无 |
| **漫画生成闭环** | 选择角色→写场景→生成漫画→下载 | 竞品❌ 不支持 |
| **跨项目复用** | 一次创建，永久复用 | 竞品⚠️ 每次重描 |

### 3.2 效果量化

| 指标 | 竞品（NovelAI等） | 我们v2 |
|------|-------------------|--------|
| 角色一致性 | ~60% | **≥90%** |
| 创建角色时间 | 10分钟手动描述 | **30秒AI提取** |
| 跨项目复用 | ❌ 不支持 | **✅ 永久复用** |
| 多角色同框 | ⚠️ 困难 | **✅ 支持** |

## 四、用户流程（基于新原型）

### 4.1 创建角色卡流程

```
上传参考图(5秒) → AI特征提取(10秒) → 确认/编辑描述(5秒) → 保存角色卡
                    ↓
              自动生成：
              • 角色缩略图
              • 特征标签（发型/服装/表情）
              • 文字描述（可编辑）
```

### 4.2 使用角色卡生成漫画流程

```
选择角色卡 → 编写场景描述 → 选择分镜数量 → 生成漫画 → 下载分享
```
