### 根本原因
- **Python 版本**: Python 3.6.8
- **当前 SQLAlchemy 版本**: 1.4.54
- **代码要求**: SQLAlchemy >= 2.0.25 (使用了 `AsyncAttrs` 等 SQLAlchemy 2.0 新特性)
- **问题**: Python 3.6 无法安装 SQLAlchemy 2.0（需要 Python >= 3.7）

**结论**: ❌ **服务无法启动**。代码与运行时环境存在 Python 版本不兼容问题。
