## 🎯 产品定位

打造一款**以漫画创作为逻辑原点**的AI动态漫创作工具，让用户通过文字剧本一键生成完整的动态漫视频。

**核心功能：** 剧本输入 → AI分镜 → 画面生成 → 动态化 → 配音配乐 → 合成输出
