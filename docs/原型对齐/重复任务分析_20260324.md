## 一、日志发现的致命Bug

### Bug 1：Dev Bot 路径重复（FileNotFoundError）

**日志证据**（dev.log）：
```
FileNotFoundError: [Errno 2] No such file or directory: '/opt/AiComic/opt/AiComic/tmp/crewai_proto_PROTO-006_7530bd9b.py'
FileNotFoundError: [Errno 2] No such file or directory: '/opt/AiComic/opt/AiComic/tmp/crewai_proto_PROTO-002_f4c6c4e4.py'
FileNotFoundError: [Errno 2] No such file or directory: '/opt/AiComic/opt/AiComic/tmp/crewai_proto_PROTO-003_a197fd7f.py'
```

**根因**：`bot_http_server_v2.py` 第479行：
```python
local_script = f"/opt/AiComic/opt/AiComic/tmp/crewai_proto_{task_id}_{script_id}.py"
```
路径前缀被写了两次，`/opt/AiComic/opt/AiComic/` 多了 `/opt/AiComic/`。

**正确应该是**：
```python
local_script = f"/opt/AiComic/tmp/crewai_proto_{task_id}_{script_id}.py"
```

**影响**：Dev Bot收到PROTO任务后，试图写入文件时立即崩溃，无法执行任何研发工作。
