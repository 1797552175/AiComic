## 一、产品概述

### 1.1 产品定位
AI创作动态漫是一款帮助用户通过AI快速生成漫画内容的工具。用户输入剧本描述，选择或创建角色，即可生成完整的漫画分镜。

### 1.2 核心价值
- **角色一致性**：基于角色卡技术，确保同一角色在不同分镜中外观一致
- **分镜控制**：支持多种镜头类型和角度
- **中文TTS**：支持中文语音合成，与画面同步

### 1.3 目标用户
- 漫画作者：快速将剧本转化为漫画
- 自媒体创作者：批量生成内容
- AI爱好者：创意表达

## 三、功能模块详述

### 3.1 故事编辑器

#### 功能说明
用户编写和管理漫画剧本，支持分章节、多角色引用。

#### 页面结构
```
┌─────────────────────────────────────────────────────────────┐
│ 故事编辑器                                    [保存] [预览] │
├───────────────┬─────────────────────────────────────────────┤
│  章节列表     │                                             │
│               │  章节标题输入框                             │
│  第1章       │                                             │
│  ├─ 第1幕   │  分镜列表                                   │
│  └─ 第2幕   │  ┌─────────────────────────────────────┐   │
│  第2章       │  │ 分镜1                              │   │
│  ├─ 第1幕   │  │ 角色选择：[角色A▼][角色B▼]        │   │
│  └─ 第2幕   │  │ 场景描述：________________        │   │
│               │  │ 对话内容：________________        │   │
│  [+ 新增章节] │  │ 镜头选择：[▼]                    │   │
│               │  └─────────────────────────────────────┘   │
│  角色关系图谱 │                                             │
│  [查看]      │  [+ 添加分镜]                              │
└───────────────┴─────────────────────────────────────────────┘
```

#### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 章节标题 | String | 是 | 最大50字符 |
| 分镜内容 | Object | 是 | 包含角色、场景、对话、镜头 |
| 角色引用 | Array | 否 | 引用的角色卡ID列表 |
| 场景描述 | String | 是 | 最大200字符 |
| 对话内容 | String | 否 | 最大500字符 |
| 镜头类型 | Enum | 是 | 静止/推进/拉远/摇镜/平移/跟随 |
| 镜头角度 | Enum | 否 | 正面/侧面/背面/特写 |

#### 接口设计

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 创建章节 | POST | /api/chapters | 创建新章节 |
| 获取章节列表 | GET | /api/chapters | 获取所有章节 |
| 更新章节 | PUT | /api/chapters/{id} | 更新章节 |
| 删除章节 | DELETE | /api/chapters/{id} | 删除章节 |
| 创建分镜 | POST | /api/panels | 创建分镜 |
| 更新分镜 | PUT | /api/panels/{id} | 更新分镜 |
| 删除分镜 | DELETE | /api/panels/{id} | 删除分镜 |

## 四、技术架构

### 4.1 系统架构
```
┌─────────────────────────────────────────────────────────────┐
│                        前端 (React)                         │
│  故事编辑器  │  角色管理  │  漫画生成  │  语音合成       │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                      后端 API (FastAPI)                      │
│  Chapter API  │  Character API  │  Comic API  │  TTS API   │
└─────────────────────────────────────────────────────────────┘
         │                                    │
         ▼                                    ▼
┌─────────────────┐              ┌─────────────────────────────────┐
│   PostgreSQL     │              │       AI 服务                    │
│   角色/章节数据  │              │  角色特征提取  │  图像生成     │
└─────────────────┘              │  TTS合成      │  LoRA加载     │
                                 └─────────────────────────────────┘
```

### 4.2 技术选型

| 组件 | 技术 | 说明 |
|------|------|------|
| 前端 | React + Ant Design | 组件库 |
| 后端 | FastAPI + Python | 高性能API |
| 数据库 | PostgreSQL | 主数据存储 |
| 图像生成 | Stable Diffusion + LoRA | 角色一致性 |
| TTS | CosyVoice / GPT-SoVITS | 中文语音 |
| 文件存储 | S3 / 阿里云OSS | 图片/音频存储 |

### 4.3 数据库表设计

```sql
-- 角色表
CREATE TABLE characters (
    id VARCHAR(36) PRIMARY KEY,
    name VARCHAR(30) NOT NULL,
    tags JSONB,
    reference_images JSONB NOT NULL,
    features JSONB,
    description TEXT,
    lora_path VARCHAR(255),
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- 章节表
CREATE TABLE chapters (
    id VARCHAR(36) PRIMARY KEY,
    title VARCHAR(50) NOT NULL,
    order_index INTEGER NOT NULL,
    story_id VARCHAR(36),
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- 分镜表
CREATE TABLE panels (
    id VARCHAR(36) PRIMARY KEY,
    chapter_id VARCHAR(36) NOT NULL,
    order_index INTEGER NOT NULL,
    character_ids JSONB,
    scene_description TEXT,
    dialogue TEXT,
    camera_type VARCHAR(20),
    camera_angle VARCHAR(20),
    generated_image_url VARCHAR(255),
    tts_id VARCHAR(36),
    created_at TIMESTAMP DEFAULT NOW()
);
```
