# 角色管理模块 技术设计文档

> 版本：v1.0 | 日期：2026-03-24 | 状态：待开发

## 一、模块概述

角色管理模块是AI创作动态漫的核心功能，负责：
1. 角色卡的创建、存储、查询
2. 从参考图提取角色特征
3. 生成角色LoRA模型

---

## 二、数据结构

### 2.1 角色实体

```python
class Character:
    id: str              # UUID
    name: str            # 角色名称
    tags: List[str]     # 标签
    reference_images: List[ReferenceImage]  # 参考图列表
    features: CharacterFeatures              # 特征描述
    description: str    # 角色简介
    lora_path: str      # LoRA模型路径
    created_at: datetime
    updated_at: datetime

class ReferenceImage:
    url: str            # 图片URL
    features: dict      # 提取的特征

class CharacterFeatures:
    hair: str           # 发型发色
    eyes: str           # 眼睛
    clothing: str        # 服装
    expression: str      # 表情
    body_type: str      # 体型
    accessories: str     # 配饰
```

### 2.2 数据库表

```sql
CREATE TABLE characters (
    id VARCHAR(36) PRIMARY KEY,
    name VARCHAR(30) NOT NULL,
    tags JSONB DEFAULT '[]',
    reference_images JSONB NOT NULL,
    features JSONB,
    description TEXT,
    lora_path VARCHAR(255),
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

CREATE INDEX idx_characters_name ON characters(name);
CREATE INDEX idx_characters_tags ON characters USING GIN(tags);
```

## 三、API接口

### 3.1 创建角色卡

**POST** `/api/characters`

Request:
```json
{
  "name": "角色名称",
  "tags": ["少年", "男主"],
  "reference_image": "base64或URL"
}
```

Response:
```json
{
  "id": "char_xxx",
  "name": "角色名称",
  "tags": ["少年", "男主"],
  "features": {
    "hair": "棕色短发",
    "eyes": "蓝色眼睛",
    "clothing": "白色T恤"
  },
  "lora_path": "/models/characters/char_xxx.safetensors"
}
```

### 3.2 获取角色列表

**GET** `/api/characters?page=1&page_size=20&search=关键词`

Response:
```json
{
  "items": [...],
  "total": 100,
  "page": 1,
  "page_size": 20
}
```

### 3.3 获取角色详情

**GET** `/api/characters/{id}`

### 3.4 更新角色卡

**PUT** `/api/characters/{id}`

### 3.5 删除角色卡

**DELETE** `/api/characters/{id}`

### 3.6 提取角色特征

**POST** `/api/characters/extract`

Request:
```json

```

Response:

## 四、核心算法

### 4.1 特征提取流程

```
参考图 → CLIP编码 → 特征向量 → LLM解析 → 结构化特征
```

1. **CLIP编码**：使用CLIP提取图像特征向量
2. **LLM解析**：将特征向量输入LLM，生成结构化描述
3. **Prompt优化**：生成可用于SD的英文prompt

### 4.2 角色一致性方案

**方案A：LoRA微调（推荐）**

```
训练流程：
1. 提取角色特征
2. 生成10-20张不同场景/动作的角色图
3. 使用SD + LoRA训练脚本微调
4. 生成角色专属LoRA模型

推理流程：
1. 加载角色LoRA
2. 输入场景描述 + 角色prompt
3. 生成保持一致性的角色图
```

**方案B：IP-Adapter**

使用IP-Adapter保持角色ID，适合快速生成。

### 4.3 LoRA训练参数

```python
LORA_CONFIG = {
    "rank": 4,           # LoRA rank，越高质量越好但越大
    "alpha": 4,          # 缩放因子
    "train_epochs": 100,
    "batch_size": 1,
    "learning_rate": 1e-4,
    "optimizer": "AdamW",
    "trigger_words": ["CHAR_NAME"]  # 触发词
}
```

## 五、文件存储

### 5.1 目录结构

```
/opt/AiComic/
├── characters/
│   ├── {char_id}/
│   │   ├── reference/         # 参考图
│   │   ├── lora/             # LoRA模型
│   │   └── thumbnails/        # 缩略图
│   └── index.json            # 角色索引
├── models/
│   └── characters/           # LoRA模型存储
│       └── {char_id}.safetensors
```

### 5.2 上传处理

```python
async def upload_reference_image(char_id: str, file: UploadFile):
    # 1. 保存原始图片
    path = f"/opt/AiComic/characters/{char_id}/reference/{uuid4()}.jpg"
    await save_file(file, path)
    
    # 2. 生成缩略图
    thumbnail_path = f"/opt/AiComic/characters/{char_id}/thumbnails/{uuid4()}_thumb.jpg"
    await generate_thumbnail(path, thumbnail_path, size=(200, 200))
    
    # 3. 提取特征
    features = await extract_features(path)
    
    return {"url": path, "features": features, "thumbnail": thumbnail_path}
```
